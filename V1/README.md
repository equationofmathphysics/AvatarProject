# Avatar Project - 你的数字工作伙伴

> 通过被动观察你的电脑行为，自动抽象你的工作模式，先做你的 Copilot，最终成为你的数字替身。

## 项目简介

Avatar Project 是一个创新的个人工作智能伙伴，它能够：

1. **观察** - 被动记录你的工作过程（屏幕、操作、应用使用）
2. **理解** - 分析你的工作模式和习惯
3. **协作** - 预测你的需求，提供智能建议
4. **执行** - 在你授权的范围内可靠执行重复任务

## 项目阶段

### Phase 1：工作记录员（当前）
- 屏幕捕获（关键帧）
- 操作记录
- 本地数据加密存储
- 工作时间轴可视化
- 每日工作摘要自动生成

### Phase 2：流程分析师
- 任务识别与切分
- 工作流模式抽取
- 决策点识别
- 模式透明展示

### Phase 3：智能协作者
- 上下文感知预测
- 智能建议系统
- 半自动执行
- 逐步信任建立

### Phase 4：可信执行者
- 授权执行机制
- 置信度驱动执行
- 安全审计
- 企业级合规

## 核心功能

### 观察引擎

观察引擎负责记录用户的工作过程，包括：

- **屏幕捕获**：关键帧捕获，不录制连续视频
- **操作记录**：应用切换、窗口变化、文件操作、URL访问
- **隐私保护**：支持隐私模式，可排除特定应用或网站

### 数据管理

- **本地加密存储**：AES-256 加密，用户数据不离设备
- **SQLite 数据库**：高效的本地数据存储
- **数据导入导出**：支持 JSON 格式的数据交换

### 工作分析

- **时间轴可视化**：清晰展示工作流程
- **应用使用分析**：统计各应用使用时长
- **每日工作摘要**：自动生成工作总结
- **重复模式识别**：发现可优化的工作流程

## 隐私与安全

我们非常重视用户隐私，采用以下安全措施：

1. **本地优先**：所有数据处理在用户设备完成
2. **数据加密**：AES-256 加密存储
3. **用户控制**：用户可随时查看、编辑、删除数据
4. **透明操作**：所有学习过程透明可追溯
5. **隐私模式**：支持排除特定应用或网站
