## 下一步：安装依赖并运行

### 1. 安装npm依赖
```bash
npm install
```

（正在后台安装中... 等待完成）

### 2. 启动开发服务器
依赖安装完成后，运行：
```bash
npm run dev
```

这将同时启动：
- Vite开发服务器（端口5173）
- Electron应用程序

### 3. 你将看到

启动成功后，你会看到：
1. 一个Electron窗口打开
2. 左侧有四个主要功能入口：
   - 工作时间轴
   - 每日工作日报
   - 工作模式识别（开发中）
   - 设置
3. 顶部有一个"开始记录"按钮
4. 界面中已经有一些模拟数据用于演示

## 常见问题

### Q: npm install 很慢怎么办？
A: 可以使用国内镜像源：
```bash
npm install --registry=https://registry.npmmirror.com
```

### Q: 项目可以运行但没有实际功能？
A: 当前版本是框架层，屏幕捕获等平台相关功能需要进一步实现。现在的版本可以用来验证UI和数据流。

### Q: 如何贡献代码？
A: 查看README.md了解更多信息。
